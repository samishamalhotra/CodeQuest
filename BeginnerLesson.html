Welcome to your Intermediate Level Lesson

Here's a list of what you need to know before playing your game:

Imagine you have a toy room filled with different things. Each thing represents a different data type in Python!

  1. Boolean (bool)

    2. It can only be True or False.
    Here's an example: 

    is_sunny = True
    is_raining = False
    Like asking: "Is the game on?" If yes, it's True, if no, it's False.


  2. Dictionaries (dict)

    3. It has keys and values (like names and their favorite colors).
    Here's an example: 

    my_dict = {"Alice": "Blue", "Bob": "Green"}
    This means Alice likes Blue and Bob likes Green.
    Another example would be comparing it to a map that tells you where to find the treasure!


  3. Float

    4. Numbers with decimals.
    Here's an example:

    pi = 3.14
    temperature = 98.6

    Imagine measuring water in a cup, like 1.5 cups!


  4. Integers (int)

    5. Whole numbers, no decimals.
    Here's an example:

    age = 7
    apples = 10

    Like counting apples or toys!

    toys.append("robot")  # You can add more toys!

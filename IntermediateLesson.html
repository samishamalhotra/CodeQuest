Welcome to your Intermediate Level Lesson

Here's a list of what you need to know before playing your game:

  1. What's a tuple?

    2. A tuple is a special way to group several pieces of information together. You can think of it like a list of items,
    but the difference is that once you create a tuple, you cannot change it (this is called "immutable").
    To create a tuple in Python: 

    my_tuple = (1,2,3,4)
    This creates a tuple with three number !!

    Remember -> once you make a tuple, you can't add / remove items from it

  2. Strings into Numbers

    3. What if you have a string like "123" and you want to treat it like a number?
    No worries! Python lets you convert strings into numbers. To do this, we use the int() function:


    number_string  "123"
number = int(number_string)
print(number)

    This will successfully turn the string "123" into the number 123 so that you can us it in math

  3. What's the length of a list?

    4. Have you ever wanted to know how many items are in a list? Python has a simple way to do that with the len() function.
    It counts how many elements are in a list or other collection.

    my_list = [1,2,3]
print(len(my_list))

    This will print 3 which is the length of my_list

  4. Creating a list

    5. A list is one of the most useful tools in Python. It’s a collection that can hold many items like numbers, words, or even other lists!
    To create a list, we use square brackets []. Here’s how:

    my_list = [1,2,3,4]

    This creates a list with four numbers. Lists can be changed, so you can add, remove, or change items as you go.

  5. Creating a set

    6. A set is another way to store multiple items, but with a twist! Unlike lists, sets don’t allow duplicates, so you can’t have the same item more than once. You create a set with set().
    Here’s how:

    my_set = set([1,2,3])

    This will create a set with the numbers 1, 2, and 3. Sets are great when you don’t want any duplicate items!
